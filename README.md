# REACT ECOMMERCE

Languages used to make the project.

-   React.js
-   Node.js

## Quick Start

# step 1

npm install in both client and server folders

```sh
$ cd server
$ npm install
```

and

```sh
$ cd client
$ npm install
```

# step 2

To start the project run npm start in server folder in one terminal and npm start in client folder in another terminal.

```sh
$ cd server
$ npm start
```

and

```sh
$ cd client
$ npm start
```
